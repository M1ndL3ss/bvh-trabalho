Instruções do editor:

- As pastas Assets/BVHScripts e Assets/BVHPrefabs contém o conteúdo que foi feito para o trabalho. O resto é parte do template Unity FPS Microgame.
- Os prefabs dentro de Assets/BVHprefabs irão mostrar gizmos com as árvores BVH se forem colocados na cena.
- Um script spawner instancia 50 prefabs, 10 de cada tipo, aleatóriamente na área de jogo quando o jogo é iniciado.

Instruções de jogo:

- WASD para mover, clique esquerdo para atirar, clique direito para mirar.
- V para mudar entre o modo BVH e o modo BruteForce.
- Observe o console para ver o tempo de processamento dos raycasts.
